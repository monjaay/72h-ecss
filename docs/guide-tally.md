# Guide Tally — Consultation des inscriptions

> **Le formulaire est déjà créé.**
> URL publique (à partager) : <https://tally.so/r/xXoX7r>
> Cette page explique uniquement comment **consulter et exporter les réponses** côté commission.

---

## 1. Se connecter

1. Aller sur **<https://tally.so>**
2. Cliquer sur **Sign in** (en haut à droite)
3. Se connecter avec le compte CCQ (email + mot de passe — voir l'admin du compte si oubli)

> Si plusieurs membres de la commission ont besoin d'accéder aux réponses, demander à l'admin du compte CCQ d'inviter chacun (`Settings → Workspace members`).

---

## 2. Ouvrir le formulaire

Dans le tableau de bord après connexion :

- Cliquer sur le formulaire intitulé **« 72H ECSS · CCQ Unité 8 »**
- Aller dans l'onglet **Submissions** (en haut, à côté de Build / Share)

Chaque ligne = une inscription. Le nombre total est affiché en haut.

---

## 3. Lire une inscription

Chaque ligne contient :

| Champ | Contenu |
| --- | --- |
| **Prénom et Nom** | Texte saisi par le participant |
| **Numéro WhatsApp** | Numéro brut — appeler/relancer si besoin |
| **Activité** | Randonnée / Don de sang / Les deux |
| **Secteur Unité 8** | Texte libre (optionnel) |
| **Date** | Horodatage de la soumission |

Cliquer sur une ligne pour voir le détail en pleine page.

---

## 4. Exporter en CSV (pour Excel)

1. Dans l'onglet **Submissions**, bouton **⋯** (trois points) en haut à droite
2. Cliquer **Export submissions** → choisir **CSV**
3. Le fichier se télécharge automatiquement
4. Ouvrir avec **Excel** ou **Google Sheets**

Astuce : faire un export **chaque soir** pendant les 72H pour avoir une copie locale sécurisée.

---

## 5. Filtrer et trier

Dans l'onglet Submissions :

- **Filtre** : icône entonnoir en haut → filtrer par activité (ex : voir seulement les inscrits au don de sang)
- **Tri** : cliquer sur l'en-tête d'une colonne pour trier (par date, par nom…)
- **Recherche** : barre en haut pour chercher un nom ou un numéro

---

## 6. Suivre les notifications en temps réel

Tally peut envoyer un email à chaque inscription :

1. Onglet **Integrations** dans le formulaire
2. Activer **Email notifications**
3. Mettre l'email principal de la commission

> Utile le jour J pour ne rater aucune nouvelle inscription tardive.

---

## 7. Statistiques rapides

Onglet **Insights** :

- Nombre total d'inscriptions
- Répartition par activité (graphique camembert)
- Évolution dans le temps (courbe)

À montrer en réunion CCQ pour faire le point.

---

## 8. Modifier le formulaire (si besoin)

Si la commission veut ajouter un champ (ex : « Tranche d'âge »), c'est **uniquement** dans l'onglet **Build** :

1. Cliquer sur **Build**
2. Glisser un nouveau champ depuis la barre de gauche
3. **Publish** en haut à droite — le formulaire public est mis à jour instantanément
4. **L'URL ne change pas** : `tally.so/r/xXoX7r` reste valide

> ⚠️ **Ne jamais supprimer le formulaire** — l'URL est imprimée sur les fiches et le QR code.

---

## 9. Problème ?

| Problème | Solution |
| --- | --- |
| Email/mot de passe perdu | `Forgot password` sur tally.so + email CCQ |
| Le lien ne s'ouvre pas chez un participant | Vérifier sa connexion, lui envoyer le QR code |
| Inscription en double | Garder la plus récente, signaler en commentaire |
| Export CSV vide | Vérifier les filtres actifs, désactiver tous les filtres |
| Autre | Contacter le WhatsApp commission `+{{WHATSAPP_CCQ}}` |
