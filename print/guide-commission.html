Guide de la commission

Tout ce qu'il faut savoir pour gérer les inscriptions et la communication numérique des 72H
1
Consulter les inscriptions
  • Aller sur tally.so et se connecter avec le compte CCQ.
  • Ouvrir le formulaire « 72H ECSS · Unité 8 ».
  • Onglet Submissions : liste des inscrits en temps réel.
  • Bouton Export → fichier CSV ouvrable dans Excel.
2
Partager le lien sur WhatsApp
  • Copier le lien : tally.so/r/xXoX7r
  • Coller dans les groupes WhatsApp Unité 8.
  • Ajouter un message court : « Inscris-toi aux 72H ECSS — 2 min, sans compte. »
  • Utiliser les 3 templates WhatsApp déjà préparés.
3
Imprimer & afficher le QR code
  • Fichier : hub/assets/qr-code.svg
  • Ouvrir dans Chrome → Ctrl+P → format A3 pour affiche.
  • Afficher devant la mosquée, l'école, les commerces clés.
  • Scanner avec le téléphone pour vérifier que ça pointe vers le bon site.
4
Fiches papier pour les non-connectés
  • Imprimer fiche-inscription.html en A5.
  • Collecter sur le terrain pendant les visites quartier.
  • Saisir manuellement dans Tally chaque soir.
  • Conserver les originaux papier pour archives CCQ.
